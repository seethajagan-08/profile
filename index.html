Python 3


Python is a high-level, interpreted & general purpose programming language. It emphasis code readability with its clear syntax & intendation structure. It was Developed by "Guido van Rossum" in the late 1989 and officially releases in 1991. It is efficient for development in Data Science, AI, Web Development, and more.


Features

  1. Open Scorce - Freely available for anyone to use
  2. Platform independent - It runs on Windows, MacOS, Linux, Android...
  3. Extensive Library ecosystem - One of the biggest reason to become famous
  4. Indented programming - Clear syntax
  5. Strong Community support
